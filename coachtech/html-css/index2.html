社員の早期離職を防ぐためには

2023-02-13 /

皆様の職場では、採用したばかりの社員さんが数年以内に離職してしまうといったことはありませんか？ここでは苦労して採用したにも関わらず数年で職場を後にしてしまう社員を増やさないために考えられる施策を記載していきます。

なぜ早期離職を招くのか

早期離職を決断してしまう理由は複数あり、中には複雑に要因が絡まって退職を決意する方もいらっしゃいます。

  • 早期離職となる原因その①　やりがいが感じられない

仕事にやりがいが感じられないというのは、人によって理由が千差万別で企業側としてもどのように対策をしたらよいかわかりかねる事柄です。

しかし、早期退職する方は本当にやりがいがないと思って退職していくのでしょうか？実際はそのほかの要因によって職場にマイナスイメージを持ってしまい、その結果目の前の仕事にやりがいが見いだせなくなってしまっていることも考えられます。もちろん仕事が本人に合わず仕事にやりがいを感じないこともありますが、かなりのケースでやりがいを感じないというのは別の要因によってもたらされていると考えられます。

考えられる対策

離職率を下げるために各企業が行っている施策は数多いですが、良く挙げられるのは1on1のミーティングがあります。これは選ばれたメンターか、その人の上司が定期的に社員と面談をしていくことです。ここで大切なのは、社員に自発的に話してもらうことです。筆者の経験としても、こうした定期的な社員へのフォローを欠いたために若手社員の不安や不満を掬えきれず早期離職を招いてしまった事案が多々あります。1on1さえすればよいわけではありませんが、こうした定期的なフォローを実施していない企業も数多いかと思いますので未実践に企業様は是非前向きに検討されるとよいかと存じます。
